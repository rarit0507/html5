태그(tag) => 요소(Element)

del, sup, sub 태그 등 크로스 브라우징 또는 크로스 미디어를 반영하지 못하는 태그는 사용하지 않는 것이 좋다.

크로스 브라우징(Cross Browsing) : 하나의 웹 문서를 여러 브라우저로 출력해보는 행위로서 모든 브라우저가 출력내용이 유사 또는 동일해야 하기 위해 하는 행위

표준 5대 브라우저 : Chrome, Safari, firefox, Opera, Edge

크로스 미디어(Cross Media) : 하나의 문서를 데스크톱, 노트북, 태블릿, 스마트폰에서 출력해보는 행위로서 해당 문서의 내용이 그 기기에 맞게 출력하는지 확인하기 위한 행위

반응형(Responsible) 디자인 : 크로스 미디어가 반영될 수 있도록 웹 페이지를 디자인하는 것.

웹 표준(Web Standard) : 정해진 문법에 맞게 웹 페이지를 작성하는 것

웹 접근성(Web Accessbility) : 시각 장애자나 노인 등 웹 환경에 취약한 계층을 위하여 모두 쉽게 웹에 접근할 수 있도록 문서를 만드는 것
